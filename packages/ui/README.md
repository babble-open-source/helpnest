# @helpnest/ui

Shared React component library used across the HelpNest dashboard and public-facing pages.

This package is private and not published to npm.

## Status

Currently a placeholder. Components will be added as they are built.

## Peer Dependencies

- `react@^18`
- `react-dom@^18`
- TypeScript 5.4

## Entry Point

`src/index.ts`

## Usage

Import components directly from the package within the monorepo:

```ts
import { ComponentName } from '@helpnest/ui'
```

## Notes

This package is workspace-internal only. Do not publish to npm.
