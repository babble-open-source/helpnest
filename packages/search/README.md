# @helpnest/search

Shared full-text and vector search utilities for the HelpNest platform.

This package is private and not published to npm.

## Status

Currently a placeholder. Implementation is planned for Phase 2.

## Planned Scope

Will wrap PostgreSQL full-text search (FTS) and pgvector (Qdrant) behind a common interface. The goal is a single search API consumed by both the web application and the SDK, abstracting the underlying search backend.

## Notes

This package is workspace-internal only. Do not publish to npm.
